# Add or update additional client ID

The **Additional Client ID** field is a reference number associated with an agreement.&#x20;

It is provided when an order is placed and, if included, appears in this field. If no number is entered at the time of ordering, you can add it manually later.&#x20;

This reference is shown on your invoices as the **PO number** to help you match them for billing and invoicing purposes.&#x20;

{% hint style="info" %}
If the number exceeds 20 characters, only the first 16 characters are shown on the invoice, followed by 3 dots (...). For more information, see [Understand your billing documents.](../../billing/understand-your-billing-documents/)
{% endhint %}

### Add or update the additional client ID

To add or update the Additional Client ID:

1. Go to **Marketplace** > **Agreements**.
2. Select the required agreement.
3. On the agreement details page, select the **Details** tab, then select **Edit**.&#x20;

<div data-with-frame="true"><figure><img src="../../../.gitbook/assets/image (946).png" alt=""><figcaption><p>Select <strong>Edit</strong> on the <strong>Details</strong> tab to update the additional client ID. </p></figcaption></figure></div>

4. In the **Edit ID** dialog, add or update the client ID, then select **Save**.&#x20;

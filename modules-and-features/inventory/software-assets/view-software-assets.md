# View software assets

The **Software Assets** page provides an overview of your active software subscriptions, current spending, and total spending.

To view software assets:&#x20;

1. Go to **Inventory** > **Software Assets**.
2. Use the search options to view your software asset inventory list. You can search using a purchase order number or reference number. You can search by purchase order number or reference number. Additionally, you can apply available filters or expand **Show more options** to access advanced filters.
3. Review details on two tabs: Software Assets and Financial Transactions.
   * The **Software Assets** tab allows you to view only your active software subscriptions and current spending. Please note that it does not show past payments.
   * The **Financial Transactions** tab provides a comprehensive view of your total spending on all software, including both subscriptions and past transactions.

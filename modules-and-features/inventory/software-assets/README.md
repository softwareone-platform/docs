# Software assets

With Software Asset Spend Management, you can oversee and manage your software asset spending across your organization. You can also export your software assets to other Software Asset Management (SAM) tools and import software assets, including those not procured through SoftwareOne.

The Software Assets module, offered as part of Cloud Managed Services, allows you to manage your software assets and inventory for software purchased through SoftwareOne.&#x20;

To access this module, select **Inventory** > **Software Assets** from the main menu.

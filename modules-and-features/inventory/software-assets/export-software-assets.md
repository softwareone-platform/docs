# Export and import software assets

The **Export** option on the **Software Assets** page enables you to download your software asset inventory in various formats.&#x20;

You can use these exports to create quick reports and organize your data in the commonly configured format (including column names and order) compatible with several Software Asset Management (SAM) tools, such as Aspera, Miss Marple, Snow, Flexera, and others.

To export software assets:

1. Go to **Inventory** > **Software Assets**.
2. Hover over the **Export** option.
3. Select your preferred export format.
4. In **Create New Report**, fill out the required fields.
5. Select **Run** to initiate the export.

{% hint style="info" %}
Depending on your SAM Tool setup, the standard reporting templates might require some adjustments. To learn about the reporting templates, see [Reports](../../other-tools/reports/).
{% endhint %}

### Import software assets

Our entitlement services allow you to import your software assets into the Marketplace Platform, even those not purchased through SoftwareOne.

For information on entitlement services and how to import your assets, [contact support](../../../help-and-support/contact-support.md).

# Copy API token

You can copy the API token from the token details page.

### Copy an API token

To copy an API token:

1. Go to **Settings** > **API tokens**.&#x20;
2. Find and select the token you want to copy. The token details page opens.
3. On the **General** tab, do the following as needed:
   * **Copy the endpoint** – Select the copy icon next to the API endpoint.&#x20;
   * **Copy the token value** – Select the copy icon next to the API token.
   * **Show or hide the token value** –  Select the visibility icon to show the token value. Select the icon again to hide it.

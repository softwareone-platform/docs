# Create API tokens

Use API tokens to access the platform programmatically. Account administrators can create and manage tokens directly in the platform.

### Create a new API token

To create a new API token:

1. Go to **Settings** > **API tokens**.&#x20;
2. Select **Add**.

<div data-with-frame="true"><figure><img src="../../../.gitbook/assets/image (990).png" alt=""><figcaption><p>Use the Add API token wizard to add a new token.</p></figcaption></figure></div>

3. Complete the steps in the **Add API token** flow:
   1. **General** – Enter the required token details, then select **Next**:
      1. **Logo** – (Optional) Upload an image or drag your file into the upload area.&#x20;
      2. **Name** – (Required) Enter a name for the token.&#x20;
      3. **Description** – (Optional) Add a description.
   2. **Modules** – Select the modules that the token can access, then select **Next**. Only modules enabled for your account are shown.&#x20;
   3. **Overview** – Review the details, then select **Add**. The token is created, and a summary is displayed.
   4. **Summary** – Select **View details** to open the token details page or select **Close**.&#x20;

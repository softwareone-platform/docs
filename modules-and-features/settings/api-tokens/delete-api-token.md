# Delete API token

If you no longer need a token, you can delete it using the **Delete** option on the **API tokens** page. Before deleting, note the following points:

* A deleted token can't be reinstated or reused.&#x20;
* Deleting a token removes it from the account, which means that the token is no longer displayed on the **API tokens** page.&#x20;

### Delete an API token

To delete a token:

1. Go to **Setting**s > **API tokens**.&#x20;
2. Find the token you want to delete.
3. Perform one of the following steps:
   * Select the token. Then, on the token details page, select the dropdown arrow <i class="fa-chevron-down">:chevron-down:</i>, and choose **Delete**.
   * On the **API tokens** page, select the actions icon (**•••**) for the token, then choose **Delete**. &#x20;

<div data-with-frame="true"><figure><img src="../../../.gitbook/assets/image (1010).png" alt=""><figcaption><p>Use the Delete option to permanently delete a token.</p></figcaption></figure></div>

4. In the **Delete API token** dialog, confirm you want to delete the token. A message is displayed stating that the API token has been deleted successfully.&#x20;

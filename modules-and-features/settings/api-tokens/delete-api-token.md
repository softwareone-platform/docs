# Delete API token

If you no longer need a token, you can delete it from the **API tokens** page.&#x20;

Before deleting a token, note the following points:

* A deleted token can't be **restored or reused.**
* Deleting a token removes it from the account,  and it no longer appears on the **API tokens** page.

### Delete an API token

To delete a token:

1. Go to **Setting**s > **API tokens**.&#x20;
2. Find the token you want to delete.
3. Do one of the following:
   * Select the token to open the token details page. Then, select the dropdown arrow <i class="fa-chevron-down">:chevron-down:</i>, and choose **Delete**.
   * On the **API tokens** page, select the actions icon (**•••**) for the token and choose **Delete**. &#x20;

<div data-with-frame="true"><figure><img src="../../../.gitbook/assets/image (1010).png" alt=""><figcaption><p>Use the Delete option to permanently delete a token.</p></figcaption></figure></div>

4. In the **Delete API token** dialog, confirm that you want to delete the token.&#x20;

A message appears confirming that the API token was deleted successfully.&#x20;

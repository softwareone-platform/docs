# Integration

The **Integration** module in the SoftwareOne Marketplace gives you access to our extensions directory containing tools you can use to enhance your workflows and customize your experience.&#x20;

Within the module, you can explore both public and private extensions that enhance workflows and extend the functionality of the SoftwareOne Marketplace. Private extensions require an invitation code that must be redeemed so that the extension can be installed, while public extensions are available directly from the extensions directory.

As an account administrator, you can install any available extension into your account. Each extension has a details page to help you understand detailed functionality, terms of service, and required permissions. After installation, you can manage which [user groups](../groups/) can see and use the installed extension.&#x20;

### Managing extensions

The **Integration** module has two main areas:&#x20;

* The **Extensions** directory, where you can browse all available extensions, view detailed information for a specific extension, and install the extension.&#x20;
* The **Installation** page, where you can see which extensions are installed, who installed them, and uninstall extensions if they are no longer needed.&#x20;

After an extension has been installed, it can appear in several predefined locations within the platform. Check the main navigation menu, relevant modules, or details pages to access the extension's features.&#x20;

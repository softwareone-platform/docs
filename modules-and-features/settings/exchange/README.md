# Exchange

The **Exchange** module in the Marketplace is a centralized location for viewing the currencies and exchange rates used for pricing and billing within the platform.

This module is read-only and is designed to provide visibility into supported currencies and their associated exchange rate configurations.

You can use the module to view all supported currencies in the platform, access detailed information about a specific currency, and view related currency pairs along with individual currency pair details.

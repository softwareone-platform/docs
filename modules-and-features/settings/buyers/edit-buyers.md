# Update buyer details

After SoftwareOne links buyers to your account, you can manage them in the platform and update buyer details as needed. You can edit information such as the buyer’s assigned contact person and the buyer’s logo.

### Update buyer details

To update the buyer details:

1. Go to **Settings** > **Buyers**.&#x20;
2. Select the buyer you want to update. The buyer details page opens.
3. Select **Edit**.

<div data-with-frame="true"><figure><img src="../../../.gitbook/assets/Buyer.png" alt=""><figcaption><p>Use the <strong>Edit</strong> option to update buyer details.</p></figcaption></figure></div>

4. In the **Edit buyer** flow, do the following as needed:
   * **General** – Add or update the buyer's logo. You can upload an image by selecting **Select file** or by dragging the image into the field.&#x20;
   * **Buyer address** – Update the buyer's address.
5. Select **Save**. The details are updated, and a confirmation message is displayed.

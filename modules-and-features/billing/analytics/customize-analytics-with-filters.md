# Customize analytics with filters

On the **Analytics** page, you can refine billing data using filters to focus on the information that matters most to you.&#x20;

By default, **Analytics** displays billing data for the last 3 months. Using filters, you can adjust the dataset to perform targeted analysis. Filters help you:

* Analyze spending for specific vendors or products.
* Compare data across different time periods.
* Apply multiple filters to narrow results.
* View billing data in a preferred currency.

All billing data is displayed according to the filters you select. When you change a filter and select **Show results**, the data updates to reflect your selection.&#x20;

### Customizing analytics with filters

To customize the data in Analytics:

1. Go to **Billing** > **Analytics**.

<figure><img src="../../../.gitbook/assets/analytics (1).png" alt=""><figcaption><p>The Analytics page within SoftwareOne Marketplace.</p></figcaption></figure>

2. In the **Filters** sidebar, apply the following customization as necessary:
   1. **Date range** - Choose a date range to define the reporting period.
   2. **Granularity** - Adjust the date granularity to enhance the data. You can choose **Daily**, **Monthly**, or **Yearly**.
      * **Daily** - Displays data for each day.
      * **Monthly** - Displays totals grouped by month.
      * **Yearly** - Displays totals grouped by year.
   3. **Group by** - Organize data using predefined categories. For example, to group by vendor, select **Vendor** and choose all the vendors you want to include. You can only choose one category.
   4. **Filter by** - Choose filters to narrow the results. For example, to filter by products, select **Product** and choose all the products you want to include. You can combine multiple filters for more refined results.
   5. **Filter conditions** - Select the desired **And** or **OR** operator.
      * **And** - If you select this operator, the results are displayed only if all filter and grouping conditions are met.
      * **Or** - If you select this operator, only one of the filters and grouping needs to be met for the results to be displayed.
   6. **Currency** - Select the currency in which you want to view billing values. By default, analytics uses your configured billing currency, but you can switch to another available currency if needed.
3. Select **Show results**.  After applying filters, both the visual chart and the table reflect your customized view.

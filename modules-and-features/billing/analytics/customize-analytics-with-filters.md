# Customize analytics with filters

On the **Analytics** page, you can refine billing data using filters to focus on the information that matters most to you.&#x20;

You can select a time period, granularity, and group data across various dimensions. The Analytics chart displays a visual summary of your top 10 entities based on your selected grouping. All remaining entities are automatically combined under a single category called Others / Unmatched to maintain a clean and readable view. You can group your data by different dimensions, including Agreement, Asset, Buyer, Client, Document type, and more.&#x20;

When you export your data to Excel, the file will include all the data that you filtered on the screen. Additionally, you can also select a target currency for your financial reporting. The system automatically applies current exchange rates to convert your data, and a note at the bottom of the page will confirm when a currency conversion has been applied.

### Customizing analytics with filters

To customize the data in Analytics:

1. Go to **Billing** > **Analytics**.

<figure><img src="../../../.gitbook/assets/analytics (1).png" alt=""><figcaption><p>The Analytics page within SoftwareOne Marketplace.</p></figcaption></figure>

2. In the **Filters** sidebar, apply the following customization as necessary:
   1. **Date range** - Choose a specific period to define the reporting timeframe.
   2. **Granularity** - Adjust how the data is grouped over time. You can choose **Daily**, **Monthly**, or **Yearly**.
      * **Daily** - Displays data for each day.
      * **Monthly** - Displays totals grouped by month.
      * **Yearly** - Displays totals grouped by year.
   3. **Group by** - Organize data by a specific category. You can choose one primary grouping category. For example, to group by vendor, select **Vendor** and choose all the vendors you want to include.&#x20;
   4. **Filter by** - Choose specific entities within categories. You can select multiple filters for more refined results.
   5. **Filter conditions** - Select the desired **And** or **OR** operator.
      * **And** - If you select this operator, the results are displayed only if all conditions are met.
      * **Or** - If you select this operator, only one condition needs to be met for the results to be displayed.
   6. **Variable -** Select **Sale price** to view your standard billing charges.
   7. **Currency** - Select the currency in which you want to view billing values. For more details, see [Currency selection and conversion](./#currency-selection-and-conversion).
3. Select **Show results**.&#x20;

After applying filters, both the visual chart and the table reflect your customized view. If you want to download your data, use the **Export** option. For more details, see [Export billing data](export-billing-data.md).

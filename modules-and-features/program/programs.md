# Programs

The **Programs** page in the platform displays all programs that are active and open for enrollment.&#x20;

By enrolling in a program, you gain access to all the benefits associated with that program. For information on the Microsoft partner programs available for enrollment through the Marketplace, see [Partner Programs](../../extensions/microsoft-cloud-solution-provider/products-and-programs/partner-programs.md).&#x20;

### Programs page

The **Programs** page displays all active programs that you can enroll in.

To access the page, choose **Program** > **Programs** from the main menu.&#x20;

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/programs-page.png" alt=""><figcaption><p>Use the Programs page to view the available programs.</p></figcaption></figure></div>

### Open program details

On the program details page, you can view detailed information for the program. Some information is read-only, while others include links you can use to navigate to further details.

To open program details:

1. Go to **Program** > **Programs**.
2. Select the program you want to view. &#x20;
3. On the details page, review the program information.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/programspage.png" alt=""><figcaption><p>Use the program details page to view additional details.</p></figcaption></figure></div>

4. Use the tabs to view additional information:

<table><thead><tr><th width="171">Tab</th><th>Description</th></tr></thead><tbody><tr><td><strong>General</strong></td><td>Shows the program requirements and enrollment details.</td></tr><tr><td><strong>Certificates</strong></td><td>Shows certificates for the program. You can also enroll in the program using <strong>Add</strong>. </td></tr><tr><td><strong>Terms</strong></td><td>Shows the terms and conditions of the program.</td></tr><tr><td><strong>Media</strong></td><td>Shows any image and video files added by the vendor or SoftwareOne.</td></tr><tr><td><strong>Documentation</strong></td><td>Shows the supporting documentation for the program.</td></tr></tbody></table>

# Complete cases

You can mark a case as **Complete** if your issue has been resolved and no further assistance is needed.&#x20;

Any user within the account can close a case, regardless of who created it. A closed case chat can also be reopened if further assistance is needed.

### Completing a case

To complete a case:

1. On the **Chats** page, select the case you want to close.
2. Select the actions icon (**•••**) in the chat window, then under **Case**, choose **Complete**.
3. In the **Complete case** confirmation, confirm that you want to complete the case.

The case closes and its status changes to **Completed**. You can reopen the case at any time. For details, see [Reopen a case](reopen-cases.md).

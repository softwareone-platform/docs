# Mobile app FAQs

### General access

<details>

<summary>Where can I get the app?</summary>

You can download the SoftwareOne Marketplace app from the Apple App Store (iOS) or Google Play Store (Android).

</details>

<details>

<summary>Can I complete the account registration in the app?</summary>

The app is for the existing users of the Marketplace Platform.&#x20;

New users must register their account through the web version at [https://portal.platform.softwareone.com/](https://portal.platform.softwareone.com/).

</details>

### Device & system requirements

<details>

<summary>What devices does the app support?</summary>

The app is optimized for mobile devices with a screen size < 7”, but it works on tablets and larger devices.

</details>

<details>

<summary>What version of iOS is supported? </summary>

The app supports iOS version 17 and later.

</details>

<details>

<summary>What version of Android is supported? </summary>

The app supports Android version 15 or later.

</details>

### App functionality

<details>

<summary>Can I purchase subscriptions and add or remove licenses using the app?</summary>

The app only supports a limited set of features optimized for mobile use.&#x20;

You cannot place orders or modify subscriptions through the app. To view the list of supported features, see [App capabilities](app-overview/#app-capabilities).

</details>

<details>

<summary>Can I update my profile through the app?</summary>

You cannot update your profile using the app.&#x20;

To update your profile, use the web version at [https://portal.platform.softwareone.com/](https://portal.platform.softwareone.com/).

</details>

<details>

<summary>Which languages does the app support?</summary>

The app supports the same languages as the web version. It uses the device language when available and falls back to English if the language is not supported.

</details>

<details>

<summary>How can I share feedback about the app?</summary>

To help us review and respond effectively, submit feedback using the **Feedback** functionality in the Marketplace Platform.&#x20;

For more details, see [Share feedback](../help-and-support/share-feedback.md#share-feedback).

</details>

### Sign-in & security

<details>

<summary>How do I sign in to the app?</summary>

Use your existing SoftwareOne Marketplace account and the verification code sent to your registered email address.&#x20;

For enhanced security, the app uses passwordless authentication. Therefore, entering a password isn't required to sign in.

</details>

<details>

<summary>My verification code is no longer valid.</summary>

Verification codes are only valid for 3 minutes. If your code is no longer valid, use the **Resend code** option during sign-in to generate a new one.

</details>

<details>

<summary>I have trouble signing in to the app.</summary>

If you have trouble signing in, try these steps:

* Check your internet connection.
* Ensure you are using the email address you provided when you set up your Marketplace account.
* Ensure that your account is active. If you haven't accepted your account invitation yet, you must do so and complete the registration first through the web version at [https://portal.platform.softwareone.com/](https://portal.platform.softwareone.com/).
* If you are not receiving the verification code, check your Spam or Junk email folders. You can also resend the code.

If the issue persists, [contact support](../help-and-support/contact-support.md) for assistance.

</details>

### Data & troubleshooting

<details>

<summary>What data do you collect?</summary>

We collect your unique user ID, which helps us identify your account, and your device ID.

Additionally, we gather usage data and diagnostic information, including crash and performance data, to improve our services.

</details>

<details>

<summary>I noticed a discrepancy in the information within the app. </summary>

If you notice any discrepancy, we recommend checking the web version of the platform.

Sometimes, temporary issues can cause discrepancies. The web version reflects the most accurate and complete data.

</details>

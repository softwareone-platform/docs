# Date Range Parameter Value

The Date Range Parameter Value is an object used in the Marketplace APIs to represent a range between two dates. This object has two fields.

<table><thead><tr><th width="138">Field</th><th width="173">Type</th><th>Description</th></tr></thead><tbody><tr><td><code>start</code></td><td>string</td><td>The start date of the range.</td></tr><tr><td><code>end</code></td><td>string</td><td>The end date of the range.</td></tr></tbody></table>

### Example

{% code title="DATE RANGE PARAMETER VALUE OBJECT" lineNumbers="true" %}
```json
{
  "start": "2019-02-10",
  "end": "2023-01-01"
}
```
{% endcode %}

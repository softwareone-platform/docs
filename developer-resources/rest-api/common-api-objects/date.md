# Date

The `Date` object is used in the Marketplace APIs to represent a single calendar date. This object contains only one field.

<table><thead><tr><th width="117">Field</th><th width="107">Type</th><th>Description</th></tr></thead><tbody><tr><td>date</td><td><code>string</code></td><td>A date value in ISO format (YYYY-MM-DD).</td></tr></tbody></table>

# Purchase additional Microsoft 365 licenses

This tutorial explains how to order additional licenses for an existing Microsoft 365 subscription by submitting a change order.

{% embed url="https://vimeo.com/985742107/1a72a8abed?share=copy" %}
Watch this video guide explaining how to buy more licenses for an existing 365 subscription.
{% endembed %}

### Prerequisites

Ensure that the agreement you want to use is active.

### Purchase additional licenses for a Microsoft 365 subscription <a href="#id-1.-launch-the-purchase-wizard" id="id-1.-launch-the-purchase-wizard"></a>

{% stepper %}
{% step %}
**Open the product**

To open the product:

1. Go to **Catalog** > **Products**.
2. Select the desired Microsoft 365 product, for example, **Microsoft 365 Business, Enterprise & Apps**.
3. On the details page, review the information, then select **Buy now**.&#x20;
{% endstep %}

{% step %}
**Select the agreement and update the license quantity**

In the guided purchase flow, complete the following steps:

1. **Create agreement** – Select an active agreement, then select **Next**.&#x20;
2. **Items** – In **New qty**, enter the total number of licenses required. For example, if the subscription currently has 1 license and you want to add 3 more, enter **4**. When done, select **Next**.

<div data-with-frame="true"><figure><img src="../../../../.gitbook/assets/tutorial_M365_items.png" alt=""><figcaption><p>Enter the new quantity of licenses for the subscription.</p></figcaption></figure></div>

3. **Details** – Add any reference details or comments, then select **Next**.
4. **Review order** – Review the order details, terms and conditions, and privacy statement, then select **Place order**.
5. **Summary** – Select **View details** to open the order details page or select **Close**.
{% endstep %}
{% endstepper %}

### Next steps <a href="#next-steps" id="next-steps"></a>

Once you place your order, we verify the details.

You can track your order on the order details page. The **General** tab on the order details page outlines the next steps.
